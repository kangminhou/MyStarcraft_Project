건물들이 공통적으로 가지고 있는 기능
 - 체력
 - 와이어 프레임
 - 건물들 고유의 기능들
 - 건물을 지을 때 체력에 따라 짓는 애니메이션 바뀜..
 - 건물이 다 지어질 경우 이벤트 발생..


가스의 경우 밑바탕 애니메이션이 아닌 고유의 애니메이션으로 시작함..


건물들의 기능들
 - 건물을 띄울 수 있고 띄웠을 때 움직일 수 있는 건물 ( 움직임 구현 )
 - 유닛을 생산할 수 있는 건물 ( 유닛 생성 구현 )
 - 연구를 할 수 있는 건물 ( 연구 구현 )
 - 공격을 할 수 있는 건물 ( 공격 구현 )
 - 자신에게 붙일 수 있는 건물 ( 붙음 구현 )
 - 계속해서 애니메이션이 동작하는 건물 ( 애니메이션 구현 )


건물들 리스트 ( 테란 )
 1. 커맨드 센터
	- 필요로 하는 건물 : 없음

	- 체력 : 1500
 	- 커맨드 센터를 다 지어질 경우 생기는 일 : 최대 인구수 +10
