Scv 정리



* 유닛이 공통적으로 가지고 있는 데이터..
	- 체력 : 60
	- 방어력 : 0
	- 이동 속도 : 2.8125
	- 시야 : 1
	- 공격 범위 : 1
	- 유닛 타입 : 지상 유닛, 생체 유닛, 기계 유닛


* 무기
	- 지상 : Fusion Cutter
	- 공중 : None


* 생산 관련
	- 생산 시 소요 시간 : 12.5
	- 필요로 하는 건물 : 없음
	- 유닛을 생성하는 건물 : Command Center
	- 생성 시 소모하는 자원		미네랄 : 50	가스 : 0	인구 수 : 1


* 필요한 기능 : 자원 채취, 주변 탐색, 공격, 건물 짓기

* 고유 기능
 1. 자원 채취
	- 광물 산지에서 미네랄을 간혈천에서 가스를 캐서 커맨드 센터에 저장할 수 있다..

 2. 건물 짓기
	- 건물을 지을 수 있는 곳에 건물을 건설할 수 있다..
	- 건물을 지을 때는 움직일 수 없으며, 맞아도 상태가 바뀌지 않고, 계속 하는 일을 한다..
