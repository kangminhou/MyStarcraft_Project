Tank 정리

* 유닛이 공통적으로 가지고 있는 데이터..
	- 체력 : 150
	- 방어력 : 1
	- 이동 속도 : 2.25
	- 시야 : 10 ( 시즈 모드 시 12 )
	- 공격 범위 : 7 ( 시즈 모드 시 12 )
	- 유닛 타입 : 지상 유닛, 기계 유닛

* 무기
	- 지상 : Arclite Cannon ( 시즈 모드 시 Arclite Shock Cannon )
	- 공중 : None


* 생산 관련
	- 생산 시 소요 시간 : 31.2
	- 필요로 하는 건물 : Machine Shop
	- 유닛을 생성하는 건물 : Factory
	- 생성 시 소모하는 자원		미네랄 : 150	가스 : 100	인구 수 : 2

* 필요한 기능 : 주변 탐색, 공격, 모드 전환

* 고유 기능
 1. 모드 전환 : 일반 모드일 경우는 시즈 모드로 시즈 모드일 경우는 일반 모드로 바뀐다. ( 바뀌면서 애니메이션 실행 및 주위의 적을 감지 못한다. )
